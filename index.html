HTMLinterest

- современный справочник по html. Для тех кто начал изучать язык. Здесь вы найдете описание всех актуальных html тегов с примерами, а также краткое описание в ячейках с элементами.

html (HyperText Markup Language)  — стандартизованный язык разметки, который позволяет создавать и структурировать веб-страницы. Называется он языком разметки, а не программирования, потому что в нем нет функций, переменных, операторов и других аспектов характерных для языков программирования. Сам код состоит из тегов — команд, указывающих браузеру какой контент отображать на странице, из этих элементов состоит сайт, даже визуально вы можете разделить страницу на блоки, скорее всего вы обнаружите как раз визуализацию тегов.

Но сам по себе html выглядит на странице не презентабельно, это просто преимущественно черно-белый сайт, где большинство элементов расположены сверху вниз, в общем такой сайт никому не нужен, ведь его просто-напросто будет не удобно читать. Тут нам на помощь приходит css — язык стилей, с помощью которого оформляют веб-сайты, он определяет например такие параметры как цвет элемента, его расположение, размеры и другое.
