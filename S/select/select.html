атрибуты

name=""

Задает уникальное имя выпадющего списка.

required

Указывает поле обязательным для заполнения.

disabled

Делает поле недоступным для взаимодействия.

autofocus

Делает автофокус на поле при загрузке страницы.

multiple

Представляет возможность выбора нескльких вариантов.

size=" "

Указывает сколько строк будет видно в окне скролла. Только при использовании атрибута multiple.
