Теги table

table - создание таблицы. Для создания строки используется тег tr, для ячейки заголовка th, для обычной ячейки td.

<table cellpadding="5em" border="2em">
<tr>
<th> α </th>
<th> 0 </th>
<th> 30 </th>
<th> 45 </th>
<th> 60 </th>
<th> 90 </th>
</tr>
<tr>
<th> sinα </th>
<td> 0 </td>
<td> 1/2 </td>
<td> √2/2 </td>
<td> √3/2 </td>
<td> 1 </td>
</tr>
<tr>
<th> sinα </th>
<td> 0 </td>
<td> √3/2 </td>
<td> √2/2 </td>
<td> 1/2 </td>
<td> 0 </td>
</tr>
</table>

атрибуты

align=" "

Выравнивание таблицы.
start - слева
center - по центру
end - справа

background=" "

Фоновое изображение.

bgcolor=" "

Цвет фона таблицы.

border=" "

Толщиина рамки таблицы.

bordercolor=" "

Цвет рамки.

cellpadding=" "

Расстояние от рамки до содержимого.

cellspasing=" "

Расстояние между ячейками.

cols=" "

Число колонок таблицы.

frame=" "

Указывает как отображать границы.
void - без отображения
border - вокруг таблицы
above - граница по верхнему краю таблицы
below - по нижнему крю таблицы
hsides - только горизонтальные границы
vsides - только вертикальные границы
rhs - граница только справа
lhs - граница только слева

height=" "

Высота таблицы.

width=" "

Ширина таблицы.

rules=" "

Выравнивание таблицы.

summary=" "

Выравнивание таблицы.
