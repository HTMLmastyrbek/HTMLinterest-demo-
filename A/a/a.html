Тег a

a - тег ссылка. Используется для перехода на другую страницу этого сайта, на страницу другого сайта, также можно использовать как якорь для перемещения на сайте.

<a href=" "> link </a>
link

стили по умолчанию

a{
color: #0000FF;    //зависит от браузера
text-decoration: underline;
cursor: pointer;
}

атрибуты

href=""

Тег имеет атрибут href="" для указания пути ссылки, он обязателен во всех версиях html кроме html5 - последней на данный момент. Значения, которые может принимать атрибут:
https://site-name.ru - На другой сайт.
site.html - На другую страницу сайта.
#id-el - Якорь, перемещение к элементу, id которого указано.
mailto:necto@mail.ru - Быстрое сообщение по почте.
+76665554433 - Звонок.

download

Указывает браузеру не переходить по заданному адресу, а скачать файл.

target=""

Указывает браузеру где открыть ссылку. Имеет значения:
_blank - В новом окне браузера.
_self - В текущем окне.
_parent - Открывает документ в родительском окне. Если родителя нет, параметр будет вести себя как _self.
_top - Открывает ссылку в окне высшего родителя, если родителя нет, то работает как значение _self.
