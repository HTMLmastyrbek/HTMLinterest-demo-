Тег base

base - создает основной адрес для относительных ссылок. Тег может быть только один на весь документ.

<base href="https://com.ru"> </base>
<a href="/css/css.html"> https://com.ru/css/css.html </a>
https://com.ru/css/css.html

атрибуты

href=""

Для указания основного адреса для относительных ссылок.

target=""

Указывает браузеру где открыть ссылку. Имеет значения:
_blank - В новом окне браузера.
_self - В текущем окне.
_parent - Открывает документ в родительском окне. Если родителя нет, параметр будет вести себя как _self.
_top - Открывает ссылку в окне высшего родителя, если родителя нет, то работает как значение _self.
