Тег audio

audio - добавляет аудио файл на сайт.

<audio src="Korol_i_SHut_-_Lesnik.mp3" controls> </audio>

стили по умолчанию

audio{
    width: 300px;    //зависит от браузера
    height: 54px;    //зависит от браузера
}
audio:not([controls]){
    display: none !important;
}

атрибуты

controls

Атрибут позволяет пользователю управлять воспроизведением, перемоткой и другими настройками.

src=""

Указывает путь к аудио-файлу.

autoplay

Автовоспроизведение аудио при открытии страницы.

preload

Загрузка аудио-файла вместе с загрузкой страницы.
auto - автоматическая загрузка при открытии страницы.
metadata - загрузка при старте воспроизведения аудио.
none - не загружать файл.

loop

Зацикливание воспроизведения аудио.

muted

Отключение звука при загрузке аудио-файла.
