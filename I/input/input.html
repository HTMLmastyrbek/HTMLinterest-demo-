атрибуты

type=" "

Задает тип поля ввода.
text - текстовове поле.
email - поле для ввода почты.
checkbox - несолько вариантов ответов (флажки).
radio - один вариант ответа (радио).
number - ввод цифр.
url - поле для ввода адреса.
button - кнопка.
password - пароль.
tel - ввод телефона
file - загрузка файла.
hidden - скрытое поле.
range - выбор значений от и до.
date - выбор даты.
time - выбор времени.
datetime - дата и время.
search - поле для поиска.
submit - кнопка отправки формы.
reset - очистить значения формы.

name=" "

Уникальное имя инпута. Используется для получения данных на сервере.

value=" "

Значение внутри поля.

maxlenght=" "

Задает максимальное количество символов.

min=" "

Минимальное значение. Для range, number, date.

max=" "

Максимальное значение. Для range, number, date.

checked

Отмечает поле radio или checkbox по умолчанию.
