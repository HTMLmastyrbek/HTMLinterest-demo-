Тег dialog

dialog - создание диалогового модального или немодального окна. Модальные окна преостанавливают работу на странице, немодальные или поп-апы позволяют взаимодействовать со всей страницей.
Для отображения элемента используется JavaScript: .showModal - для отображения окна как модального, .show - для отображения поп-апа.

<button id="open-pop-up">
открыть поп-ап
</button>
<button id="open-modal">
открыть модалку
</button>
<dialog id="pop-up">
Это поп-ап
<button id="close-pop-up">
закрыть
</button>
</dialog>
<dialog id="modal">
Это модальное окно
<button id="close-modal">
закрыть
</button>
</dialog>

const PopUp = document.getElementById('pop-up');
const ClosePopUpBtn = document.getElementById('close-pop-up');
const OpenPopUpBtn = document.getElementById('open-pop-up');

OpenPopUpBtn.addEventListener("click", () => {
PopUp.show();
});
ClosePopUpBtn.addEventListener("click", () => {
PopUp.close();
});

const Modal = document.getElementById('modal');
const CloseModalBtn = document.getElementById('close-modal');
const OpenModalBtn = document.getElementById('open-modal');

OpenModalBtn.addEventListener("click", () => {
Modal.showModal();
});
CloseModalBtn.addEventListener("click", () => {
Modal.close();
});
Это поп-ап
Никак не влияет на работу сайта. Можно использовать для вспомогательных уведомений, опросов и т.п..
Это модальное окно
Модальное окно блокирует все действия на сайте. Как правило используется для подтверждения чего либо, важных уведомлений.

атрибуты

open

По умолчанию открывает диалоговое окно. Окно открывается как поп-ап.
